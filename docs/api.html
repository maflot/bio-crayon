Core Module

Core BioCrayon class for managing biological data colormaps.

class bio_crayon.core.BioCrayon(source=None, require_metadata=False)[source]

Bases: object

Main class for managing biological data colormaps.

Supports loading colormaps from files, URLs, or dictionaries, and provides methods for accessing and converting colormaps.
